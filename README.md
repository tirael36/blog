# Tirael — Video Games & Technology Blog

A simple blog for gaming & tech, with authentication, roles, and a rich text editor (admin/author only).

## Tech Stack

- Next.js (React)
- Tailwind CSS
- Supabase (auth, database)
- React Quill (editor)

## Setup

1. **Clone & Install**

```bash
git clone https://github.com/yourname/tirael-blog.git
cd tirael-blog
npm install
```

2. **Configure Supabase**

- Create a project at [Supabase](https://supabase.com).
- Get your project’s `anon` key and URL, set as environment variables:

```bash
NEXT_PUBLIC_SUPABASE_URL=your-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```

- In Supabase SQL editor, create tables:

```sql
-- Users (profiles)
create table profiles (
  id uuid primary key references auth.users(id),
  role text not null default 'reader'
);

-- Posts
create table posts (
  id bigint generated by default as identity primary key,
  title text not null,
  content text not null,
  author_id uuid references profiles(id),
  author_email text,
  created_at timestamp default now()
);
```

- Set up Row Level Security (RLS) as needed.

3. **Run**

```bash
npm run dev
```

## Roles

- `admin`, `author` — can create posts
- `reader` — can only view

Edit user roles in the `profiles` table.
